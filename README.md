(필수)
- [ ] 개인메모

- [ ] 좋아요 기능

- [ ] Public/Private 설정
- [ ] 사용자별 테두리 및 테마 색상 변경 기능
- [ ] 조회 기능

- Framework : Django(2.2), Vue.js, django-restframework
